# CPG Synergy — Website Redesign

5 unique design variants for the CPG Synergy marketing platform.

## Designs
1. **The Authority** — Dark luxury power funnel
2. **The Trust Builder** — Light social-proof funnel
3. **The Closer** — Bold urgency-driven funnel
4. **The Storyteller** — Editorial narrative funnel
5. **The Minimalist** — Clean direct-response funnel

## Getting Started
```bash
npm install
npm start
```

## Deploy to Vercel
```bash
git init
git add .
git commit -m "CPG Synergy redesign — 5 design variants"
git remote add origin YOUR_GITHUB_REPO_URL
git push -u origin main
```
Then connect the repo to Vercel for automatic deployments.

## Design Switcher
Use the floating toolbar at the bottom of the page to switch between designs.
